# Arcade Cabinet Parameters

- Power Consumption: The electrical cost to keep the cabinet running.
- Popularity: A metric determining how attractive a cabinet is to players. The higher the popularity, the more likely players are to use it. This value decays over time.
- Condition (Health): A value between 0% and 100% indicating the machine's state. This value decreases as the machine is used. Once it hits a certain threshold (30%), the machine enters maintenance (Out of Order) and cannot be used. After a set period of time, the machine returns to 100% Condition.
- Price per Credit: Once a machine is purchased, the user can decide how much it costs to play ($1, $2, etc.). This parameter, along with Popularity, influences whether a user decides to play and spend money.
- Difficulty Dip Switches: Historically, owners could physically change the machine's difficulty. This serves as a management mechanic: increase difficulty to generate more revenue or decrease it to keep customers happier.

# Staff (Personnel)

- Janitor: Manages a value from 0% to 100% indicating how clean the arcade is. This affects whether players visit the arcade or not.
- Maintenance Technician: The person who fixes the machines when they enter Upkeep / Servicing mode.
- Ghostbuster: Occasionally, ghosts appear in the arcade.

# Concession Stand / Snack Bar

- F&B (Food and Beverage): Food/Drinks.
- Fountain Drinks / Soda Fountain: Soft drinks from a tap.
- Milkshakes: Shaked dairy drinks.
- Chips (US) / Crisps (UK): Bagged potato chips.
- Vending Machine: Automated snack dispensers.
- Candy: Sweets and treats.
